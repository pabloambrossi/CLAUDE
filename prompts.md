# Prompts para Claude

Colección de prompts y plantillas útiles para usar con Claude.

---

## Resumen de textos

```
Resume el siguiente texto en 5 puntos clave:

[TEXTO]
```

## Corrección de escritura

```
Corrige la gramática, ortografía y estilo del siguiente texto. 
Explica brevemente cada cambio:

[TEXTO]
```

## Explicación de conceptos

```
Explícame [CONCEPTO] como si tuviera 10 años.
Luego dame una explicación más técnica para un nivel avanzado.
```

## Generador de ideas

```
Dame 10 ideas creativas para [TEMA].
Sé específico y original. Evita las ideas genéricas.
```

## Análisis de pros y contras

```
Analiza los pros y contras de [TEMA/DECISIÓN].
Organízalos en una tabla y termina con tu recomendación.
```

---

> 📌 **Tip:** Cuanto más contexto le das a Claude, mejores son las respuestas.
